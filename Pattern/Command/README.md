# Command pattern
メソッドの呼び出しを具現化するパターンの特徴
1. メソッドの呼び出しをクラス化して、インスタンスを入れ替えることで処理を動的に切り替えることができるようにする。
2. 依存性の注入でメソッドを汎用化できる
3. 基本的には`execute()`,`undo()`を備える
